# RealBuy

An e-commerce web application built with Django.

## Features
- User registration and login
- Product listing
- Shopping cart
- Checkout process
- Order management

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap

## Installation

git clone https://github.com/LawalNawas/realbuyshop.git

cd realbuyshop

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
